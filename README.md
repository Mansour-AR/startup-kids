# Startup Kid 🚀

A single-file, browser-playable HTML/JS/CSS game that teaches youth entrepreneurship (ages 10–16) through an interactive simulation.

You start summer break with **$50** and 8 weeks to turn it into a real micro-business — choosing what to sell, how to price it, how to market it, and how to handle the unexpected.

## Play it

Just open **[`index.html`](index.html)** in any modern browser. No build step, no backend, no install. The only external dependency is Chart.js (loaded from a CDN) for the end-of-game skill chart.

## What it teaches

- **12 business ideas** with distinct economics (products vs. services, thick vs. thin margins).
- **An 8-week business curriculum** — one real concept per week: Capital → Profit Margin → Customer Acquisition → Break-even → Cash Flow → Reinvestment → Reputation → Scaling.
- **Authentic tradeoffs** in every weekly decision: pricing, marketing, reinvest-vs-save, time/energy, and customer service. No choice is strictly "correct."
- **Four skills** that visibly level up: Pricing, Marketing, Customer Service, Money Management.
- **A debrief** with a skills radar chart, personalized takeaways from your actual choices, and a reflection prompt.

Three non-punitive endings (Thriving Entrepreneur / Steady Learner / Tough Lesson) keep failure framed as a learning moment. A full session takes about 5–15 minutes.

## Tech

- Single HTML file, inline CSS + JS, fully client-side.
- Responsive — plays on desktop and mobile.
- In-memory state with a restart button.
